# Vijayvenkatesh Jayaraj — Portfolio

A recruiter-focused data analytics portfolio built with Next.js 15, TypeScript, and Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

Open the local URL printed by Next.js.

## Quality checks

```bash
npm run lint
npm run build
```

## Content updates

Portfolio copy is centralized under `src/data`. Update contact and social settings in `src/data/profile.ts`. Phone display is controlled by `showPhone` and is off by default.

Place the real resume at `public/Vijayvenkatesh_Jayaraj_Resume.pdf` so all Resume buttons download it.
